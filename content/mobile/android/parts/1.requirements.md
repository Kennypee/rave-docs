## **Requirements**

Before we proceed to integrate rave, there are a few requirements and settings you should observe:

### The minimum supported SDK version is 15

When setting up your Android project, you should select and SDK version 15 and above otherwise you wonâ€™t be able to install the rave Android SDK.
